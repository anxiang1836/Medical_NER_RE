病科,上海市内分泌代谢病研究所,上海市内分泌代谢病临床医学中心
对“阿仑膦酸钠疗效与维生素 D 血清水平的
相关性冶一文的编后评论
刘建民
本期发表了潘凌等[1] 《阿仑膦酸钠疗效与维生素
D 血清水平的相关性》的文章,研究认为绝经后骨质疏
松妇女服用阿仑膦酸钠治疗期间,血清 25-羟维生素 D
(25-OHD)水平应该维持在 50 nmol / L 以上。 虽然该
文在研究对象选择、研究设计、药物疗效评定和数据分
析等方面都需要进一步探讨和完善,但该文的结果再
次说明在骨质疏松药物治疗中需要关注患者的维生素
D 状态。
国内外的流行病学研究都一致说明人群中维生素
D 不足的状况非常普遍。 对北京和上海共计 3 200 多
名 50 ~ 70 岁男性和女性的横断面调查显示,维生素 D
缺乏 ( 25-OHD < 50 nmol / L) 和不足 ( 50 nmol / L ≤
25-OHD< 75 nmol / L ) 的 比 例 分 别 高 达 69. 2% 和
24. 4% [2]。 维生素 D 不足与骨质疏松、骨质疏松性骨
折,糖尿病、心脑血管疾病和肿瘤等多种疾病风险有
关[3],适当补充维生素 D 对于充分发挥抗骨质疏松药
物升高骨密度、降低骨折风险的疗效是非常有帮助
的[4],因此目前各国指南中都把钙和维生素 D 作为防
治骨质疏松的基本措施。
双膦酸盐是一种骨吸收抑制剂,能有效治疗骨质
疏松症。 在双膦酸盐治疗骨质疏松时,同时补充维生
素 D 是否能发挥协同作用尚无肯定的结论。 其中的
原因是多方面的,药物治疗疗程、所用维生素 D 的剂
量、患者的顺应性、所观察指标(骨密度,骨生化或骨
折)的不同,以及血清 PTH 水平的高低等都是干扰因
素。 有研究发现与单独服用阿仑膦酸盐的绝经后骨质
疏松妇女相比,阿仑膦酸盐加维生素 D 组治疗后的血
清 PTH 没有升高,骨吸收指标 CTX 下降更为明显。
该研究同时发现基线 25-OHD 水平超过 20 ng / ml(50
nmol / L)的骨质疏松妇女即使是单独补充阿仑膦酸
盐,其治疗后血清 CTX 的下降程度也要高于 25-OHD
水平低于 20 ng / ml 者[5]。 之前一项来自日本的研究
提示只有当血清 25-OHD 水平高于 25 ng / ml (62. 5
nmol / L), 双膦酸盐治疗才能取得满意的升高腰椎骨
密度的效果[6]。 目前一般认为治疗骨质疏松时,血清
25-OHD 的理想水平是 75 nmol / L[7]。
不管血清 25-OHD 的最佳切点是多少,这些研究
和本刊潘凌等的文章都说明至少对于维生素 D 水平
低下的骨质疏松人群,在双膦酸盐治疗过程中,提高血
清 25-OHD 水平有助于进一步加强双膦酸盐的疗效。
